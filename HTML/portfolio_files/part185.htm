Exchange-traded fund A portfolio of stocks and other investable assets that is traded as a single security on an organized exchange. Trading an exchange- traded fund is similar to trading any other stock listed on the exchange.

Expected return The expected mean value of a stock return or a portfolio return when the return is considered a random variable.
