8.1 INTRODUCTION

In the two preceding chapters we established the framework for factor models, our main quantitative equity portfolio management (QEPM) tools for assessing stock returns and risk. As we have seen, one idea underlies all factor models: The average stock return equals the product of factor premiums and factor exposures. We have shown how to build both the fundamental factor model and the economic factor model on this idea using past data and esti- mates based on the data. While the model is built from past data, though, it runs on future values because managers want to predict future stock returns.
