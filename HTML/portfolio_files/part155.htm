Short Returns


When a portfolio has short positions, either in an actual portfolio or a model portfolio, the performance analyst will have to compute the short returns to calculate the overall performance of the portfo- lio. Although this is not a complicated task, it is different from long-only positions. Suppose, for example, that the portfolio has a short position in stock A from period t to t+k. The daily short returns will be computed as the minus of the daily long returns. These returns can be used to compute any statistics, such as aver- age returns, correlations with other indices, and other basic statis-

tics. Thus, if stock A’s daily returns are rt,t+1, rt+1,t+2, and so on, then the short returns of stock A are rS,t,t+1 = —rt,t+1, rS,t+1,t+2 = —rt+1,t+2, and so on. You can use these to compute the daily market value of the entire short position and thus combine it with any other posi- tions to find the market value of the entire portfolio.

However, for computing the performance of the portfolio over a longer period, say, from t to t + k, one cannot use geometric link- ing as is done with long positions. That is,

These new weights can be used on the next day’s returns to compute the overall portfolio return for each day. One might also notice that, unlike long positions, positive returns from a short position lead to a smaller weight in the overall portfolio. These daily portfolio returns are useful for computing various statistics about the volatility of the portfolio; however, the performance analyst should remember that yet again these daily portfolio returns cannot be geometrically linked to get the return for the whole portfolio. In that case, the long returns of all the short positions should be taken over the entire holding period, and then the negative of those returns used. The weights should be the starting weights at the beginning of the holding period.
