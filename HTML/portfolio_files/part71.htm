8.4 MODEL-BASED FORECAST

Forecasting is not usually a portfolio manager’s forte. It is a chal- lenging task even for highly quantitatively oriented portfolio man- agers, and, in any case, it is not the best use of their time. However, there may be an in-house model that seems to forecast a particular factor very well. In this case, the portfolio could benefit from time spent on forecasting.
