On Outliers



An outlier is an observation in a distribution that is very extreme and has an extremely small probability of occurring. An outlier is something that looks very inconsistent with the rest of the data. These outliers could arise as a result of data errors by the data sup- plier, they could be due to an incorrect distributional assumption of the underlying data, or they could be a novel phenomenon. If they arise from data errors, it is a good idea for the QEPM to remove these data entirely. If they arise from an incorrect distribu- tional assumption, this is usually difficult to determine; thus the QEPM may wish to truncate the values to reasonable values so as to not distort the portfolio creation process (e.g., to 3 or —3). If they are a novel phenomenon but accurate, the QEPM still may wish to truncate the values, just for safety. It still will be a strong signal vis-à-vis other stocks.

There are several methods to label outliers. One method is

There are various methods to accommodate or deal with out- liers in the Z-score model. One method is known as winsorization. Winsorization is the process of replacing outliers with other values in order to prevent the outliers from unduly influencing the esti- mates. One form of winsorization we discussed in the main text is to convert all Z-scores with an absolute value greater than 3 into 3 or —3. This implicitly labels outliers if absolute value is greater than 3.

Another method is known as trimming. This is the process of removing them values of consisting of n/2 of the largest values and n/2 of the smallest values. The Z-scores are then recomputed for the remaining data. A version of this is suggested in the text, which is to simply remove all Z-scores whose absolute values are greater than some determined value such as 3, 4, or 5.

There are several tests to detect outliers in a data set. Unfortunately, most of the tests require that the data come from the normal distribution. The tests are Rosner’s test, Dixon’s test, and the Grubb’s test. Rosner’s test and Grubb’s test are suitable for large data sets of 25 or more data points (the typical QEPM case), and Dixon’s test is for very small data sets of 25 or fewer observations. See Gibbons (1994) for Rosner’s and Dixon’s tests and Grubbs (1969) for Grubb’s test. For tests of outliers with nonnormal data, we refer the reader to Iglewicz (1993) and Barnett and Lewis (1984).
