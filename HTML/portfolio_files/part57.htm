The correlation bet- ween the returns on a pair of stocks consists of the correlation between their nondiversifiable components and the correlation between their diversifiable components. Instead of correlation, we will use covariance; the covariance between rit and rjt is

C(rit , rjt) = i’tV(f)jt + C(Eit, Ejt) (6.32)

In practice, we estimate only the correlation between the non- diversifiable components of two stocks’ returns. As for the diversi- fiable components, the correlation between them can, in principle, be estimated in a straightforward way using the standard estima- tion technique. In practice, though, it is rarely done this way. One reason is that there are often simply too many parameters to esti- mate precisely. If there are N stocks, then [N(N—1)]/2 covariances have to be estimated. These covariances cannot be estimated pre- cisely without using a great many time intervals for the estimation. The second reason that the standard estimation technique is hard- ly used to estimate the diversifiable component of the return is that the diversifiable return, like diversifiable risk, has little bearing on the stock return to the investor (or so says modern financial eco- nomic theory). For both these reasons, the convention is to assume that diversifiable risk C(Eit , Ejt) equals zero.
