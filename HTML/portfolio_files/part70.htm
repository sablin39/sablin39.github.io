Thus, in the fundamen- tal factor model, we can assume that the factor premium fˆestimat- ed from time periods 1, …, T will be valid for time T + 1. these parameters stay constant for one more period, time T + 1.1 Forecasting becomes a necessary step when we are using the

economic factor model from Chapter 7. The economic factor model is not dynamic, so we need to forecast the value of the explanatory variable, the factor premium. In the economic factor model, the explanatory variable (factor premium ft ) is known only at the end of time t, as is the dependent variable (stock return rit ). Thus, at the end of time T, we do not have the value of the factor premium for time T +1, and we cannot predict the factor exposure for time T + 1, namely, ri,T+1, without forecasting fT+1.
